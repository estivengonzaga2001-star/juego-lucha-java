# Juego de Lucha en Java

Proyecto desarrollado en Java utilizando Programación Orientada a Objetos.

## Tecnologías usadas

- Java 17
- Maven
- JUnit 5
- Mockito
- JaCoCo

## Patrones implementados

- Factory Pattern
- Strategy Pattern

## Funcionalidades

- Creación de personajes
- Ataques aleatorios
- Sistema de pelea
- Tests automatizados
